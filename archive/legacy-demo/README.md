# Legacy Demo 参考资产

这里收的是旧版 demo 时代留下来的参考资产。

## 包含内容

- `frontend-vite-demo/`
  - 旧的 React + Vite 单前端 demo
  - 包含 `src/`、旧路由、旧样式、旧测试和 Vercel 配置
- `docs/`
  - 旧 PRD、可视化 PRD、demo 开发计划
- `prototypes/`
  - 早期 HTML 原型
- `deliverables/`
  - 已导出的 PDF 交付件
- `bundles/`
  - 历史 zip 归档包

## 使用原则

- 这里的内容保留为参考，不再作为当前实现的主路径
- 新的一期功能请落在根目录下的 `frontend/`、`backend/`、`data/`、`docs/`
- 如果后续要复用旧 demo 的某些交互或文案，请明确“搬运到新工程”，不要继续在这里叠改
