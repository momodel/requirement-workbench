# Query Patterns

## Ask for grounded summary when

- the agent needs a source-backed overview before asking clarification questions
- multiple documents cover the same topic and need synthesis

## Ask for citations when

- the runtime is about to promote a point into a visible conclusion
- the UI needs citation chips or evidence links

## Ask for comparison when

- two source sets may disagree
- a mapping or rule seems inconsistent

## Do not over-query

Prefer one targeted question over many broad ones. Pull only the evidence needed for the current turn.
