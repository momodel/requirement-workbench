# External Notes

The project [`PleasePrompto/notebooklm-skill`](https://github.com/PleasePrompto/notebooklm-skill) is a useful external reference.

Important points from its README:

- it is designed for local Claude Code use
- it explicitly says it does not work in the web UI
- it uses browser automation
- it contrasts itself with the author's MCP server, which is positioned for broader multi-tool compatibility

That makes it good reference material for:

- browser-based NotebookLM interaction patterns
- prompt phrasing
- local skill ergonomics

It does not remove the need for this project's own:

- `EvidenceRuntime`
- `NotebookLMService`
- project-state write rules
